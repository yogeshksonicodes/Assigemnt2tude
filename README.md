# Assigemnt2tude
A responsive Laundry Services Hero Section built using HTML and CSS, featuring a modern navbar and full-screen hero layout.
This project implements a modern and responsive Hero Section for a Laundry Service web application using pure HTML and CSS. It includes a navigation bar with essential links, a full-screen hero layout using viewport units, and a call-to-action button. The design is clean, user-friendly, and optimized to fit the screen without scrolling. Ideal for beginners practicing UI layout and responsive design.
